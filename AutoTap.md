# Contrato de Privacidad y Términos de Uso - Auto Tap

**Última actualización:** 5 de marzo de 2025

Este contrato describe el compromiso de **PatSaint** con la privacidad y seguridad de los usuarios de la aplicación **Auto Tap**.

## 1. Compromiso de No Recopilación de Datos
Auto Tap ha sido diseñada como una herramienta de utilidad pura. Declaramos explícitamente que:
- **NO** recopilamos ni almacenamos información personal del usuario.
- **NO** tenemos acceso a tus cuentas, contraseñas, fotos o archivos.
- **NO** compartimos ni vendemos datos a terceros. Toda la configuración de tus "clips" permanece localmente en tu dispositivo.

## 2. Uso de la API de Accesibilidad (AccessibilityService)
Nuestra aplicación utiliza la API de Accesibilidad de Android única y exclusivamente para cumplir su función principal:
- **Propósito:** Simular toques (taps) en los puntos de la pantalla que el usuario ha configurado previamente.
- **Privacidad:** El servicio de accesibilidad de Auto Tap NO lee lo que escribes, NO observa tus conversaciones y NO extrae información de las aplicaciones que utilizas. Su funcionamiento es ciego y solo inyecta clics en coordenadas X/Y.

## 3. Informes de Errores
Para garantizar que la app no se cierre, el sistema puede generar informes técnicos anónimos en caso de error (crashes). Estos informes solo contienen datos del modelo de celular y el código del error, nunca datos personales.

## 4. Sin Fines de Lucro y Transparencia
Auto Tap es una herramienta de automatización sencilla. No se lucra con el rastreo de usuarios ni con el análisis de comportamiento.
